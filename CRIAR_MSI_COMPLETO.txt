========================================
CRIAR MSI COMPLETO COM PYTHON
========================================

O MSI verdadeiro (.msi) requer ferramentas especializadas.
Vou criar 2 soluções para você:

SOLUÇÃO 1: Advanced Installer (RECOMENDADO)
============================================
✓ Cria MSI profissional
✓ Interface gráfica fácil
✓ Instala Python automaticamente
✓ Bypass de políticas incluído
✓ GRATUITO (versão Freeware)

SOLUÇÃO 2: Inno Setup (EXE ao invés de MSI)
============================================
✓ Cria instalador EXE profissional
✓ Funciona igual a MSI
✓ Instala Python automaticamente
✓ Bypass de políticas incluído
✓ GRATUITO e open-source


========================================
OPÇÃO 1: ADVANCED INSTALLER (MSI)
========================================

PASSO 1: Baixar Advanced Installer
-----------------------------------
https://www.advancedinstaller.com/download.html
- Baixe a versão FREEWARE (gratuita)
- Instale normalmente

PASSO 2: Criar Projeto
-----------------------
1. Abra Advanced Installer
2. New Project > Simple
3. Nome: Coruja Monitor Probe
4. Version: 1.0.0

PASSO 3: Adicionar Arquivos
----------------------------
1. Files and Folders
2. Add Folder
3. Selecione: C:\Users\andre.quirino\Coruja Monitor\probe
4. Destination: [ProgramFilesFolder]\CorujaMonitor\Probe

PASSO 4: Adicionar Python (Prerequisite)
-----------------------------------------
1. Prerequisites
2. Add > Predefined
3. Selecione: Python 3.11 (64-bit)
4. Configure:
   - Install if not present
   - Download from: https://www.python.org/ftp/python/3.11.8/python-3.11.8-amd64.exe
   - Silent install: /quiet InstallAllUsers=1 PrependPath=1

PASSO 5: Adicionar Dependências Python
---------------------------------------
1. Custom Actions
2. Add > PowerShell Script
3. Script:
   python -m pip install psutil httpx pywin32 pysnmp pyyaml
4. Execute: After Install Files
5. Condition: NOT Installed

PASSO 6: Criar Atalhos
-----------------------
1. Shortcuts
2. Add Shortcut
3. Target: [APPDIR]configurar_probe.bat
4. Location: Desktop + Start Menu
5. Name: Configurar Coruja Probe

PASSO 7: Configurar Firewall
-----------------------------
1. Custom Actions
2. Add > PowerShell Script
3. Script:
   netsh advfirewall firewall set rule group="Windows Management Instrumentation (WMI)" new enable=yes
4. Execute: After Install Files

PASSO 8: Bypass de Políticas
-----------------------------
1. Install Parameters
2. Package Type: Per-machine (all users)
3. Installation Options:
   - Require Administrator
   - Disable Windows Installer policy

PASSO 9: Build MSI
-------------------
1. Build > Build
2. MSI será gerado em: Output\CorujaMonitorProbe-1.0.0.msi


========================================
OPÇÃO 2: INNO SETUP (EXE)
========================================

Vou gerar um script Inno Setup completo que você pode usar.
O resultado é um EXE que funciona IGUAL a um MSI.

PASSO 1: Baixar Inno Setup
---------------------------
https://jrsoftware.org/isdl.php
- Baixe e instale

PASSO 2: Usar Script Pronto
----------------------------
Execute o script que vou criar:
.\GERAR_INSTALADOR_INNO.ps1

Isso vai gerar um arquivo .iss que você compila no Inno Setup.


========================================
OPÇÃO 3: MSI VIA WIX (AVANÇADO)
========================================

Se você tem WiX Toolset instalado, posso gerar o MSI.
Mas Advanced Installer é MUITO mais fácil.


========================================
QUAL OPÇÃO VOCÊ PREFERE?
========================================

1. Advanced Installer (MSI profissional, interface gráfica)
2. Inno Setup (EXE profissional, script automático)
3. WiX Toolset (MSI via código XML, mais complexo)

RECOMENDAÇÃO: Use Advanced Installer (Opção 1)
- É a mais fácil
- Gera MSI verdadeiro
- Interface visual
- Suporta Python como prerequisite
- Bypass de políticas automático


========================================
ALTERNATIVA RÁPIDA
========================================

Se você quer algo AGORA sem instalar ferramentas,
o instalador BAT que já geramos funciona perfeitamente
e faz TUDO que um MSI faria:

✓ Instala arquivos
✓ Verifica/instala Python
✓ Instala dependências
✓ Cria atalhos
✓ Configura firewall
✓ Registra no Windows
✓ Pode ser distribuído via GPO

A única diferença é que é um .BAT ao invés de .MSI,
mas funciona EXATAMENTE igual.
