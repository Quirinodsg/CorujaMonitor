========================================
🔧 PYTHON NÃO ESTÁ SENDO INSTALADO - SOLUÇÃO
========================================

PROBLEMA:
O instalador não está instalando o Python automaticamente


========================================
✅ SOLUÇÃO 1: SCRIPT SEPARADO (RECOMENDADO)
========================================

Criamos um script só para instalar o Python:

1. Vá para a pasta: probe-installer
2. Execute: INSTALAR_PYTHON.bat (como Administrador)
3. Aguarde 2-5 minutos
4. Depois execute: INSTALAR_TUDO.bat


========================================
✅ SOLUÇÃO 2: INSTALAR MANUALMENTE
========================================

1. Baixe Python 3.11.8:
   https://www.python.org/ftp/python/3.11.8/python-3.11.8-amd64.exe

2. Execute o instalador

3. IMPORTANTE - Marque estas opções:
   ✅ "Add Python to PATH"
   ✅ "Install for all users"

4. Clique em "Install Now"

5. Aguarde instalação

6. Depois execute: INSTALAR_TUDO.bat


========================================
✅ SOLUÇÃO 3: USAR CHOCOLATEY
========================================

Se você tem Chocolatey instalado:

choco install python311 -y


========================================
🔍 VERIFICAR SE PYTHON FOI INSTALADO
========================================

Abra CMD e execute:

python --version

Deve mostrar: Python 3.11.8

Se não mostrar:

"C:\Program Files\Python311\python.exe" --version

========================================
⚠️ PROBLEMAS COMUNS
========================================

PROBLEMA: "Falha ao baixar Python"
SOLUÇÃO: Baixe manualmente e coloque na pasta

PROBLEMA: "Python não encontrado" (após instalação)
SOLUÇÃO: Reinicie o CMD ou execute:
set PATH=%PATH%;C:\Program Files\Python311

PROBLEMA: Instalação trava
SOLUÇÃO: Aguarde 5 minutos, pode estar instalando

PROBLEMA: Antivírus bloqueia
SOLUÇÃO: Desabilite temporariamente


========================================
📞 SUPORTE
========================================

Se nada funcionar:

1. Verifique o log: python_install.log
2. Instale Python manualmente
3. Execute só a parte de dependências:

python -m pip install psutil httpx pywin32 pysnmp pyyaml
