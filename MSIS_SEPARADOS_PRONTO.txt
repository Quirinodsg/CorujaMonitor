========================================
🚀 COMO GERAR OS MSIS
========================================

REQUISITOS:
✅ WiX Toolset v3.11
   Download: https://github.com/wixtoolset/wix3/releases

✅ Windows PowerShell

✅ Permissões de Administrador

COMANDO:
PS> .\GERAR_MSIS_SEPARADOS.ps1

RESULTADO:
• installer/output/SetupDependencias.msi (~30 MB)
• installer/output/SetupProbe.msi (~2 MB)


========================================
📋 ORDEM DE INSTALAÇÃO
========================================

1️⃣ SetupDependencias.msi
   • Instala Python 3.11.8
   • Instala dependências
   • Tempo: 3-5 minutos

2️⃣ SetupProbe.msi
   • Copia arquivos
   • Cria atalhos
   • Tempo: 1 minuto

3️⃣ Configurar
   • Desktop > Configurar Coruja Probe
   • IP: 192.168.31.161
   • Token: [fornecido pelo admin]


========================================
✨ VANTAGENS DOS MSI SEPARADOS
========================================

✅ MODULAR:
   • Instala só o que precisa
   • Atualiza componentes separadamente
   • Python pode já estar instalado

✅ PROFISSIONAL:
   • Aparece no Painel de Controle
   • Desinstalação limpa
   • Interface gráfica de instalação

✅ FLEXÍVEL:
   • Reinstala só a Probe sem Python
   • Atualiza Python sem reinstalar Probe
   • Menor tamanho individual

✅ COMPATÍVEL:
   • Funciona em AD e Workgroup
   • Sem erro 8235
   • Instalação silenciosa disponível

✅ MANUTENÇÃO:
   • Atualização seletiva
   • Diagnóstico mais fácil
   • Controle granular


========================================
🎯 CASOS DE USO
========================================

CASO 1: Instalação Nova
→ SetupDependencias.msi
→ SetupProbe.msi
→ Configurar

CASO 2: Python Já Instalado
→ Pular SetupDependencias.msi
→ SetupProbe.msi
→ Configurar

CASO 3: Atualizar Probe
→ Desinstalar SetupProbe.msi
→ Instalar nova versão
→ Configuração mantida

CASO 4: Atualizar Python
→ Desinstalar SetupDependencias.msi
→ Instalar nova versão
→ Probe continua funcionando

CASO 5: Reinstalar Tudo
→ Desinstalar ambos
→ Instalar SetupDependencias.msi
→ Instalar SetupProbe.msi
