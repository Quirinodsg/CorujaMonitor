================================================================================
  SE DEMORAR MUITO (mais de 5 minutos)
================================================================================

1. Pressionar Ctrl+C para cancelar

2. Verificar logs:
   docker logs coruja-nginx
   docker logs coruja-frontend
   docker logs coruja-api

3. Reiniciar se necessário:
   docker-compose -f docker-compose.yml -f docker-compose.https.yml restart

4. Ou parar e iniciar novamente:
   docker-compose -f docker-compose.yml -f docker-compose.https.yml down
   docker-compose -f docker-compose.yml -f docker-compose.https.yml up -d
