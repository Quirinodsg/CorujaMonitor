  ✅ CORREÇÃO: MOVER SERVIDORES ENTRE GRUPOS
═══════════════════════════════════════════════════════════════

PROBLEMA RESOLVIDO:
- Não era possível mover servidores para grupos existentes
- Campo de grupo era texto livre (tinha que digitar manualmente)
- Ao criar servidor, não tinha opção de escolher grupo

SOLUÇÃO IMPLEMENTADA:
✓ Dropdown com grupos existentes no modal de edição
✓ Dropdown com grupos existentes no modal de criação
✓ Campo adicional para criar novo grupo
✓ Opção "Sem grupo" para remover servidor de grupo

═══════════════════════════════════════════════════════════════
  COMO USAR - MOVER SERVIDOR PARA GRUPO EXISTENTE
═══════════════════════════════════════════════════════════════

1. NO DASHBOARD (http://192.168.31.161:3000)
   - Menu "Servidores"
   - Localizar o servidor que deseja mover
   - Clicar no botão "✏️" (Editar) ao lado do servidor

2. NO MODAL DE EDIÇÃO:
   - Campo "Grupo / Empresa": Dropdown com grupos existentes
   - Selecionar o grupo desejado (ex: "Servidores Techbiz")
   - OU selecionar "Sem grupo" para remover de qualquer grupo
   - Clicar em "Salvar"

3. CRIAR NOVO GRUPO:
   - No mesmo modal, campo "Ou criar novo grupo"
   - Digite o nome do novo grupo
   - Clicar em "Salvar"
   - O servidor será movido para o novo grupo

═══════════════════════════════════════════════════════════════
  COMO USAR - ADICIONAR SERVIDOR JÁ EM UM GRUPO
═══════════════════════════════════════════════════════════════

1. NO DASHBOARD:
   - Menu "Servidores" → "+ Adicionar Servidor"

2. PREENCHER DADOS DO SERVIDOR:
   - Probe, Tipo, Nome, IP, Protocolo, etc.

3. CAMPO "GRUPO / EMPRESA":
   - Dropdown com grupos existentes
   - Selecionar grupo desejado
   - OU deixar "Sem grupo"
   - OU criar novo grupo no campo abaixo

4. CLICAR EM "Adicionar Dispositivo"

═══════════════════════════════════════════════════════════════
  ALTERAÇÕES NO CÓDIGO
═══════════════════════════════════════════════════════════════

ARQUIVO: frontend/src/components/Servers.js

1. ESTADO INICIAL DO newServer:
   - Adicionado campo: group_name: ''

2. MODAL DE EDIÇÃO (handleEditServer):
   - Substituído input text por dropdown
   - Dropdown lista todos os grupos existentes
   - Campo adicional para criar novo grupo

3. MODAL DE CRIAÇÃO (handleAddServer):
   - Adicionado seção com dropdown de grupos
   - Campo adicional para criar novo grupo

4. FUNÇÃO handleAddServer:
   - Adicionado group_name no payload do POST
