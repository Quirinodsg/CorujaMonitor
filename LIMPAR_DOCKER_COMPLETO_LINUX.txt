═══════════════════════════════════════════════════════════════
  EXPLICAÇÃO DOS COMANDOS
═══════════════════════════════════════════════════════════════

1. docker-compose down --volumes --remove-orphans
   → Remove containers, networks, volumes e órfãos

2. docker system prune -af --volumes
   → Remove TUDO: imagens, containers, volumes, cache
   → -a = todas as imagens (não só as não usadas)
   → -f = força (sem confirmação)
   → --volumes = remove volumes também

3. systemctl restart docker
   → Reinicia daemon do Docker

4. docker-compose up -d
   → Recria tudo do zero
