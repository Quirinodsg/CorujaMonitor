🐛 PROBLEMAS COMUNS:

┌──────────────────────────────────────────────────────────────────────────────┐
│ Problema: Aba Segurança não aparece                                         │
│ Solução: Limpe o cache do navegador (Ctrl + Shift + R)                      │
│          Ou abra em aba anônima (Ctrl + Shift + N)                          │
│                                                                              │
│ Problema: Erro ao salvar configurações                                      │
│ Solução: Verifique se está logado como admin                                │
│          Verifique logs: docker-compose logs api --tail 50                  │
│                                                                              │
│ Problema: Configurações não persistem                                       │
│ Solução: Execute: docker-compose exec api python migrate_auth_config.py    │
│                                                                              │
│ Problema: Containers não estão rodando                                      │
│ Solução: Execute: docker-compose up -d                                      │
