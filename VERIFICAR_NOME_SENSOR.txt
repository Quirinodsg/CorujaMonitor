========================================
VERIFICAR NOME EXATO DO SENSOR
========================================

Execute no servidor Linux:

docker-compose exec postgres psql -U coruja -d coruja_monitor -c "SELECT id, name, sensor_type, server_id FROM sensors WHERE sensor_type = 'disk' ORDER BY name;"

Isso vai mostrar todos os sensores de disco e seus nomes exatos.

Depois use o nome EXATO para deletar:

docker-compose exec postgres psql -U coruja -d coruja_monitor -c "DELETE FROM metrics WHERE sensor_id = [ID_DO_SENSOR];"

docker-compose exec postgres psql -U coruja -d coruja_monitor -c "DELETE FROM incidents WHERE sensor_id = [ID_DO_SENSOR];"

docker-compose exec postgres psql -U coruja -d coruja_monitor -c "DELETE FROM sensor_notes WHERE sensor_id = [ID_DO_SENSOR];"

docker-compose exec postgres psql -U coruja -d coruja_monitor -c "DELETE FROM sensors WHERE id = [ID_DO_SENSOR];"

Substitua [ID_DO_SENSOR] pelo ID que aparecer na consulta.
