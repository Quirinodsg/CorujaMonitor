================================================================================
ENVIAR PARA GIT (Git Bash no NOTEBOOK):
================================================================================

git add docker-compose.yml
git commit -m "Adicionar restart always nos containers Docker"
git push origin master

================================================================================
APLICAR NO SERVIDOR LINUX (via PuTTY):
================================================================================

# 1. Atualizar código do Git
cd /home/administrador/CorujaMonitor
git pull origin master

# 2. Recriar containers com nova configuração
docker-compose down
docker-compose up -d

# 3. Verificar que containers estão com restart policy
docker ps --format "table {{.Names}}\t{{.Status}}\t{{.Ports}}"

# 4. Verificar restart policy
docker inspect coruja-api | grep -A 3 RestartPolicy
docker inspect coruja-postgres | grep -A 3 RestartPolicy

# 5. Testar API
curl http://localhost:8000/health
