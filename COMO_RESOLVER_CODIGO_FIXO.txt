  COMO RESOLVER: Google Authenticator com Código Fixo
================================================================================

PROBLEMA:
O Google Authenticator está mostrando sempre o mesmo número e não está 
rotacionando a cada 30 segundos.

DIAGNÓSTICO:
✅ Servidor funcionando corretamente
✅ TOTP gerando códigos corretamente
✅ QR Code no formato correto
❌ Problema: QR Code antigo ou duplicado no Google Authenticator

================================================================================
  SOLUÇÃO RÁPIDA (3 opções)
================================================================================

OPÇÃO 1: Script Interativo (RECOMENDADO)
-----------------------------------------
Execute no PowerShell:

    .\resolver_mfa_codigo_fixo.ps1

O script irá guiá-lo passo a passo pelo processo de correção.


OPÇÃO 2: Solução Manual Rápida
-------------------------------
1. No Google Authenticator:
   - Remova TODAS as contas "CorujaMonitor"

2. No PowerShell:
   - Execute: .\desabilitar_mfa_todos.ps1

3. No navegador (http://localhost:3000):
   - Login → Configurações → Segurança
   - Clique em "Habilitar MFA"
   - Escaneie o NOVO QR Code
   - Verifique se o código está MUDANDO a cada 30s
   - Ative o MFA

4. Teste o login com MFA
