========================================
✅ OPÇÃO 1 - INSTALAÇÃO COMPLETA
========================================

Recomendado para primeira instalação!

O que faz:
1. ✅ Verifica se Python está instalado
2. ✅ Baixa Python 3.11.8 (25 MB) se necessário
3. ✅ Instala Python automaticamente
4. ✅ Instala dependências (psutil, httpx, pywin32, pysnmp, pyyaml)
5. ✅ Copia arquivos da probe
6. ✅ Configura firewall (WMI)
7. ✅ Cria atalhos (Desktop + Menu Iniciar)
8. ✅ Registra no Windows

Após concluir: Volta ao menu automaticamente


========================================
🔧 OPÇÃO 2 - INSTALAR APENAS PYTHON
========================================

Use quando:
• Python não está instalado
• Quer atualizar Python

O que faz:
1. Verifica se Python já existe
2. Se não existir, baixa Python 3.11.8
3. Instala automaticamente
4. Adiciona ao PATH do Windows

Após concluir: Volta ao menu


========================================
📦 OPÇÃO 3 - INSTALAR DEPENDÊNCIAS
========================================

Use quando:
• Python já está instalado
• Faltam bibliotecas

Requer: Python instalado

O que faz:
1. Atualiza pip
2. Instala psutil (monitoramento)
3. Instala httpx (comunicação)
4. Instala pywin32 (APIs Windows)
5. Instala pysnmp (SNMP)
6. Instala pyyaml (configuração)

Após concluir: Volta ao menu


========================================
📁 OPÇÃO 4 - INSTALAR PROBE
========================================

Use quando:
• Python e dependências já instalados
• Quer reinstalar apenas a probe

O que faz:
1. Copia arquivos Python (.py)
2. Copia coletores (collectors/)
3. Copia scripts BAT
4. Copia documentação
5. Configura firewall (WMI)
6. Cria atalhos no Desktop
7. Cria atalhos no Menu Iniciar
8. Registra no Windows

Instalado em: C:\Program Files\CorujaMonitor\Probe

Após concluir: Volta ao menu


========================================
🔍 OPÇÃO 5 - VERIFICAR INSTALAÇÃO
========================================

Use para:
• Testar se tudo está OK
• Diagnosticar problemas

O que verifica:
1. ✅ Python instalado e versão
2. ✅ Dependências (testa cada uma)
3. ✅ Arquivos da probe
4. ✅ Status geral do sistema

Mostra:
• [OK] - Tudo certo
• [AVISO] - Algo faltando
• [ERRO] - Problema encontrado

Após concluir: Volta ao menu

========================================
🔧 SOLUÇÃO DE PROBLEMAS
========================================

ERRO: "Falha ao baixar Python"
SOLUÇÃO: 
1. Baixe manualmente:
   https://www.python.org/ftp/python/3.11.8/python-3.11.8-amd64.exe
2. Coloque na mesma pasta do instalador
3. Execute novamente (opção 1 ou 2)

ERRO: "Python não encontrado"
SOLUÇÃO:
1. Use opção 2 do menu (Instalar Python)
2. Ou instale manualmente e marque "Add Python to PATH"

ERRO: "Dependências faltando"
SOLUÇÃO:
1. Use opção 3 do menu (Instalar Dependências)
2. Ou use opção 5 para ver qual está faltando

ERRO: "Arquivos não encontrados"
SOLUÇÃO:
1. Use opção 4 do menu (Instalar Probe)
2. Verifique se a pasta "probe/" existe
